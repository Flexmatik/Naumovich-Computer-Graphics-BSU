# Отчет по Лабораторной работе №3: Базовые растровые алгоритмы

## 1. Реализованные алгоритмы
В работе представлены следующие алгоритмы:
1.  **Пошаговый:** Простой алгоритм через уравнение прямой.
2.  **ЦДА (DDA):** Инкрементальный алгоритм с использованием float.
3.  **Брезенхем (Отрезок):** Целочисленный, быстрый.
4.  **Брезенхем (Окружность):** Использование симметрии.

### Бонусные алгоритмы
5.  **Сглаживание Ву (Wu's Algorithm):** Вычисляет интенсивность двух пикселей в столбце в зависимости от того, насколько близко к их центрам проходит идеальная линия. Устраняет "лесенку".
6.  **Кривая Безье (Quadratic):** Построение гладкой кривой по трем точкам с использованием алгоритма Де Кастельжо.
